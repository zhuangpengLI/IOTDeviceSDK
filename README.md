# IOTDeviceSDK
## 为设备上云提供接入开箱即用sdk
- 支持mqtt,coap,modbus,cwmp(tr069),usp(tr369),snmp等协议 
- 支持Arduino，ESP-IDF，RaspberryPi，合宙等各种硬件平台 
- 支持openwrt，RT-Thread等OS
- 减少重复开发，快速接入各种云平台
## 项目目录
* platform-generic --------------------- 适配通用平台
* platform-fastbe ----------------------- 适配蜂信物联网平台
